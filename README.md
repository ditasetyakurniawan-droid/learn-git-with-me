# Latihan Git

Repository ini dibuat untuk latihan Git bagi pemula. Repository ini berisi beberapa file sederhana yang dapat digunakan untuk mempraktikkan perintah-perintah Git dasar.

## Perintah Git Dasar

Berikut adalah beberapa perintah Git dasar yang dapat dipraktikkan:

1. `git init` - Inisialisasi repository Git baru
2. `git status` - Melihat status perubahan pada repository
3. `git add` - Menambahkan file ke staging area
4. `git commit` - Menyimpan perubahan ke repository
5. `git log` - Melihat history commit
6. `git branch` - Melihat dan membuat branch baru
7. `git checkout` - Berpindah antar branch
8. `git merge` - Menggabungkan branch
9. `git pull` - Mengambil dan merge perubahan dari remote repository
10. `git push` - Mengirim perubahan ke remote repository

## Latihan yang Disarankan

1. Clone repository ini
2. Buat branch baru
3. Lakukan perubahan pada salah satu file
4. Commit perubahan tersebut
5. Merge branch baru ke branch dev
6. Push perubahan ke remote repository

## Struktur Repository

- `README.md` - File ini, berisi informasi tentang repository
- `index.html` - File HTML sederhana
- `style.css` - File CSS untuk styling
- `script.js` - File JavaScript untuk interaktivitas
- `data.txt` - File teks untuk latihan perubahan sederhana
- `contributors.md` - Daftar kontributor project
